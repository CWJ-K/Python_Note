# Writing Conventions

> Naming convention: using questions or long words help fast understand the whole picture instead of short words

* Introduction
  * why I write this note
* Fundamental Concepts
  * concepts without commands
  * pictures
* Packages (Optional)
  * introduction of packages
  * package name
* Commands 
  * Package 1
  * Package 2
